# Electric_measure_sensor
an electrical network parameter monitoring device based on an annular current sensor
Система мониторинга параметров электрической сети с помощью кольцеового трансформатора тока ACX-1050 на основе микроконтроллера ES8266 NodeMCU v3.

Цели:
- мониторинг действующих значений тока и потребляемой мощности в рельном времени
- синхронизация измерений с реальным временем
- хранение данных о потреблении ввиде csv таблиц с почасовой иксацией рассчитываемых величин (среднеквадратичный тока за час, средняя потреблённая мощность за час, минимум и максимум тока за час)
- отображение данных в web интерфейсе:
  * действующие значения тока, мощность,
  * график нагрузки за последние 60 минут,
  * суточный график нагрузки за предыдущие сутки,
  * таблица средних значений за прошедшие 5 дней (мощность потреблённая за сутки, минимум тока нагрузки и момент(время) измерения, максимум тока нагрузки и момент(время) измерения)
